== Description ==

Blocks Collection plugin is a collection of page building premium blocks for the new Wordpress Gutenberg block editor. Building wordpress pages with a block editor combined with this plugin gives you more control to customize your page the way you want.

This plugin currently includes following blocks:

> Pricing Block

Kindly note this plugin is currently in it's early stages and ever-evolving, so we will be continuously adding more premium blocks for FREE in near future. The plugin is modular so you can anytime disable the blocks which you don't require.

== Installation ==

1. Upload 'blocks-collection' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Done!
